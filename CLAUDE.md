## Git / Push

Para fazer push ao repositório GitHub, configurar o remote com o token pessoal de acesso (PAT) do proprietário do repositório:

```
git remote set-url origin https://<TOKEN>@github.com/invisestore-sudo/InviseStore.git
```

O token já está configurado no remote local desta sessão. Branch principal: `main`.
